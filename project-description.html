PXMart DB 權限申請系統 — 專案說明

內容說明 · 專案背景及使用的 AI 工具

專案背景

本專案為全聯實業（PXMart）內部開發的「DB 權限申請系統」，用於取代傳統紙本或郵件申請流程，讓員工可透過網頁介面線上申請資料庫存取權限，並由 IT 審核人員線上進行審核與簽核。

系統採前後端整合架構，前端以 Vue 3 + Element Plus 建置多步驟申請精靈，後端以 Spring Boot 3 + SQLite 提供 REST API，審核完成後自動寄送通知信，並可產生正式 PDF 申請單供存檔。

為提升使用者體驗，系統內建 AI 客服助理，讓使用者無需翻閱文件即可即時了解申請流程與系統現況。

使用的 AI 工具

工具 用途
Claude Code Anthropic 輔助系統全端開發，包含後端 Spring Boot API、前端 Vue 3 元件、資料庫設計、PDF 產生邏輯等程式碼的撰寫與除錯。
claude-opus-4-8 Claude API · Anthropic 整合於系統內建的「DB 權限助理」聊天機器人，即時查詢申請統計資料（待審核／已核准／已拒絕筆數），以自然語言回答使用者關於申請流程、進度查詢等問題。
精簡版說明

本系統為 PXMart 內部 DB 權限申請平台，採用 Vue 3 + Spring Boot 全端架構。開發過程全程以 Claude Code 輔助撰寫程式碼；系統上線後整合 Claude API（claude-opus-4-8） 作為 AI 客服助理，能即時讀取資料庫申請狀態，以繁體中文自然語言回答使用者的申請相關問題。
